# A Normal Guessing Game
This is A Normal Guessing Game. The objective is to guess a random number between 1-100 in as few guesses as possible.

## Important
Project URL: https://roadmap.sh/projects/number-guessing-game

To run this project, you must have Python installed on your computer. You can install Python here: https://www.python.org/downloads/

## How to run the project
### Without Git
1. Download the ZIP file (can be accessed via the green <> Code button on my repository)
2. Extract the full ZIP file
3. Enter a terminal (e.g., the Command Prompt and Visual Studio Code) and run one of these lines if you're on Windows: 

py main.py

python main.py

If you are using macOS or Linux, run:

python3.main.py

### With Git
Enter the following lines through a terminal (e.g. Visual Studio Code and PyCharm):

git clone `https://github.com/MicroChip818/a-normal-guessing-game.git`

cd a-normal-guessing-game

py main.py

Alternatively, you may run python main.py on Windows or python3 main.py on macOS and Linux.

## Difficulties
Easy: 7 maximum guesses

Medium: 5 maximum guesses, 1 hint

Hard: 3 Maximum guesses, 2 hints

## Hints
Hints are offered if the difficulty is Medium or Hard. Hints may be used at any time or not at all, but keep in mind that they are limited per round.

Types of hints offered are whether the number is odd or even, the sum or difference of its digits, and whether the number is a multiple of a number from 3-10 or a prime number.

Hints are selected at random, and do not repeat in a round.

## Menu (requires at least 1 round played)
After playing your first round, you can view your total statistics or stats of a certain difficulty before playing another round or exiting the game.
